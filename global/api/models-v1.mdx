Moonshot-v1 is the classic text generation model series. The only difference between variants is the maximum context length; there is no difference in capability.

## Model Variants

| Model | Context | Description |
|-------|---------|-------------|
| `moonshot-v1-8k` | 8,192 tokens | Suitable for short text generation |
| `moonshot-v1-32k` | 32,768 tokens | Suitable for long text generation |
| `moonshot-v1-128k` | 131,072 tokens | Suitable for very long text generation |
| `moonshot-v1-auto` | — | — |
| `moonshot-v1-8k-vision-preview` | 8,192 tokens | Vision model |
| `moonshot-v1-32k-vision-preview` | 32,768 tokens | Vision model |
| `moonshot-v1-128k-vision-preview` | 131,072 tokens | Vision model |

## Parameter Defaults

| Parameter | Default |
|-----------|---------|
| `temperature` | 0.0 |
| `top_p` | 1.0 |
| `n` | 1 (max 5) |
| `presence_penalty` | 0 (modifiable) |
| `frequency_penalty` | 0 (modifiable) |
