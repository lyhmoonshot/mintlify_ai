kimi-k2 is a Mixture-of-Experts (MoE) foundation model with exceptional coding and agent capabilities, featuring 1 trillion total parameters and 32 billion activated parameters.

## Model Variants

| Model | Context | Description |
|-------|---------|-------------|
| `kimi-k2-0905-preview` | 256k | Enhanced Agentic Coding, frontend code quality, and context understanding |
| `kimi-k2-0711-preview` | 128k | Base version |
| `kimi-k2-turbo-preview` | 256k | High-speed version |
| `kimi-k2-thinking` | 256k | Thinking model with multi-step tool calls and reasoning |
| `kimi-k2-thinking-turbo` | 256k | High-speed thinking version, balancing deep reasoning and speed |

## Parameter Defaults

| Parameter | kimi-k2 series | kimi-k2-thinking series |
|-----------|---------------|------------------------|
| `temperature` | 0.6 | 1.0 |
| `top_p` | 1.0 | 1.0 |
| `n` | 1 (max 5) | 1 (max 5) |
| `presence_penalty` | 0 (modifiable) | 0 (modifiable) |
| `frequency_penalty` | 0 (modifiable) | 0 (modifiable) |

<Note>
When `temperature` is close to 0, `n` can only be 1. Otherwise, the API returns `invalid_request_error`.
</Note>

## Supported Features

- Tool Use (Function Calling)
- JSON Mode
- Partial Mode

<Warning>
kimi-k2 series **does not support** vision capabilities (image/video understanding). Use kimi-k2.5 for vision tasks.
</Warning>
