Moonshot-v1 是经典的文本生成模型系列，不同变体的区别仅在于最大上下文长度，效果上并无差异。

## 模型变体

| 模型 | 上下文 | 说明 |
|------|--------|------|
| `moonshot-v1-8k` | 8,192 tokens | 适用于生成短文本 |
| `moonshot-v1-32k` | 32,768 tokens | 适用于生成长文本 |
| `moonshot-v1-128k` | 131,072 tokens | 适用于生成超长文本 |
| `moonshot-v1-auto` | — | — |
| `moonshot-v1-8k-vision-preview` | 8,192 tokens | Vision 视觉模型 |
| `moonshot-v1-32k-vision-preview` | 32,768 tokens | Vision 视觉模型 |
| `moonshot-v1-128k-vision-preview` | 131,072 tokens | Vision 视觉模型 |

## 参数默认值

| 参数 | 默认值 |
|------|--------|
| `temperature` | 0.0 |
| `top_p` | 1.0 |
| `n` | 1（最大 5） |
| `presence_penalty` | 0（可修改） |
| `frequency_penalty` | 0（可修改） |
