kimi-k2 是一款具备超强代码和 Agent 能力的 MoE 架构基础模型，总参数 1T，激活参数 32B。

## 模型变体

| 模型 | 上下文 | 说明 |
|------|--------|------|
| `kimi-k2-0905-preview` | 256k | 增强 Agentic Coding 能力、前端代码美观度和上下文理解 |
| `kimi-k2-0711-preview` | 128k | 基础版本 |
| `kimi-k2-turbo-preview` | 256k | 高速版本 |
| `kimi-k2-thinking` | 256k | 长思考模型，支持多步工具调用与思考 |
| `kimi-k2-thinking-turbo` | 256k | 长思考高速版，兼顾深度推理与速度 |

## 参数默认值

| 参数 | kimi-k2 系列 | kimi-k2-thinking 系列 |
|------|-------------|----------------------|
| `temperature` | 0.6 | 1.0 |
| `top_p` | 1.0 | 1.0 |
| `n` | 1（最大 5） | 1（最大 5） |
| `presence_penalty` | 0（可修改） | 0（可修改） |
| `frequency_penalty` | 0（可修改） | 0（可修改） |

<Note>
当 `temperature` 接近 0 时，`n` 只能为 1，否则将返回 `invalid_request_error`。
</Note>

## 支持的功能

- 工具调用（Tool Use / Function Calling）
- JSON Mode
- Partial Mode

<Warning>
kimi-k2 系列**不支持**视觉功能（图片/视频理解），如需视觉能力请使用 kimi-k2.5。
</Warning>
